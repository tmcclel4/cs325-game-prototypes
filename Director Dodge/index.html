Director Dodge

  • High Concept: Do what the Snobby Director says!
  • Theme: The theme of this game is snob.
  • Mandated Variety: The mandated variety elements for this game are keyboard input, deterministic, director, and puzzle.
  • Player Experience Goals: Players should experience a fast-paced game of reflexes and memory.
  • Gameplay: The player must follow the commands from the snobby director. The commands are chosen randomly from 4 possible operations(up, left, right, down) using random functions. The commands come in three waves that consist of one, three, and then five commands. If the player presses the wrong button, too many, or not enough, they lose. Even after losing, the director continues to shout at you. Input is blocked except between waves to block unintentional inputs from the player.
  • Prototype Goal: The main mechanic being tested in this game is the random command generation and handling matching multiple inputs with those commands.
  • Features: This game features custom artwork and carefully timed events.
  • Setting/Premise: A movie is being made about an astronaut who must save Earth from an astroid.
  • Story: You are an aspiring actor trying to keep your job by pleasing your snobby director!
  • Target Audience: The target audience for this game is people who like fast-paced reflex testing games.
  • Play Time: This game takes roughly 20 seconds to play.
  • Strategies: The only strategy for this game is quickly clicking the correct keys after they are displayed.
  • Prototype Assets: All assets created by myself.
  • Prototype Closest Other Game: This prototype borrows gameplay ideas from Simon Says, but no code distribution.
