The Next Extinction

  • High Concept: Your alien race has sent a remote controlled astroid to destroy earth and it is up to you to make it to the target!
  • Theme: Astroid heading for Earth
  • Mandated Variety: This game uses pointer input, random object generation, and the play style is dodging for a set amount of time.
  • Player Experience Goals: Players should experience exhiliarating dodging of falling objects while maintaining control of an unwieldy astroid.
  • Gameplay: The player may direct an astroid through space to avoid being struck by missiles. The core mechanic of this game is the astroid accelerating towards the pointer. When a missile collides with the astroid the game is over and the screen changes. If all missiles are avoided, a success screen plays at the end of the time limit.
  • Prototype Goal: Primarily movement. Collisions as well.
  • Features: You are trying to destroy Earth by piloting an astroid instead of saving it!
  • Setting/Premise: An alien race (unnamed at this time) is attempting to eliminate mankind by sending a remote controlled astroid to recreate the dinosaur extinction event.
  • Story: The story in this game is the same as the setting. It does not have a more in-depth storyline in this version.
  • Target Audience: The target audience is any age group of people who would like to goof off for about 5-10 minutes.
  • Play Time: The max time the game can run is 15 seconds.
  • Strategies: The best strategy for the game is keeping the astroid moving in a tight circle to maintain stability and avoid slingshotting into the missles.
  • Prototype Assets: I created the missile and astroid sprites and background from scratch using microsoft paint.
  • Prototype Closest Other Game: This game borrows the most elements from Invaders and Defender.
